## 🚀 Projetos em Destaque

### 🤖 IndicaAI (SaaS)
Um pipeline automatizado construído em Python que atua como um "Personal Tech Recruiter". Ele utiliza o modelo **Google Gemini (LLM)** via API com Pydantic para ler currículos em PDF, varrer plataformas de emprego (Gupy, LinkedIn) via Web Scraping resiliente e retornar um painel em **Streamlit** apenas com as vagas de alto *match*. Arquitetura desenhada em microsserviços e orientada a Clean Code.

### 🍃 Spring Boot & MongoDB API
Desenvolvimento de uma API RESTful completa utilizando Java e Spring Boot, com persistência de dados no MongoDB. O projeto demonstra o domínio na criação de endpoints estruturados, injeção de dependências e manipulação de bancos NoSQL no ecossistema Spring.
